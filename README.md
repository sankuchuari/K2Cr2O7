# K2Cr2O7 整合包

> 一个注重性能优化、全面工业自动化、机械拓展与现代枪械的 Minecraft 整合包

K2Cr2O7 是一个专注于 **流畅体验** 与 **多维功能拓展** 的 Minecraft Forge 整合包。它在完美兼容瓦尔基里天空物理引擎的基础上，通过机械动力（Create）、通用机械（Mekanism）、沉浸工程（IE）与应用能源2（AE2）的深度交织，配合 KubeJS 自动化脚本生态，构建出了一个支持跨星系探索、硬核枪械防卫与极致现代工业自动化的硬核生存世界。

---

## ✨ 特性亮点

* ⚡ **性能深度优化**：全方位覆盖 FPS 增强、超分辨率渲染、带宽网络优化、无缝加载与异步粒子渲染，保障中低配玩家的流畅体验。
* 🌿 **世界群系与未知探索**：内置 Alex 的洞穴、超多生物群系与暮色森林，支持搭建宇宙飞船前往外太空、建造现代化村庄或深入地底深渊。
* 🚗 **多维重工业与物理引擎**：不仅包含机械动力，更融合了瓦尔基里天空物理引擎（含太空、竞赛附加）。你可以建造真正能飞行的飞艇、战舰和移动工厂。
* ⚙️ **极致自动化流水线**：整合通用机械（Mekanism）、沉浸工程、AE2 及海量跨模组联动。支持自动烹饪、全自动矿石开掘、流体燃烧发电以及现代化跨区块物流。
* 💻 **KubeJS 脚本魔改**：引入完善的 JavaScript 脚本生态前置，支持全方位的配方融合、事件监听与工艺线深度定制。
* 🔫 **现代硬核枪械战斗**：永恒枪械工坊与 Apex 枪械工艺等模组带来的逼真射击手感，搭配自动化火炮、防御铁丝网和防御炮塔，让基地防卫战更具策略性。

## 🧭 入门指南与任务路线

### 🔰 第一步：开局、拼音查配方与生存

* 创建新世界，建议开启【超多生物群系】与【Lios Overhauled Villages】带来的新世界村庄。
* 初始目标：找到村庄、建立基地，使用 **[通用拼音搜索]** 在 **[JEI物品管理器]** 中快速查阅庞大的配方库。
* 推荐探索与农业：
* [Xaero 小地图](https://www.mcmod.cn/class/1701.html) / [Xaero 世界地图](https://www.mcmod.cn/class/1483.html)
* [农夫乐事](https://www.mcmod.cn/class/2820.html) 与 [洞穴乐事](https://www.google.com/search?q=https://www.mcmod.cn/class/19055.html)：开启农业自动化与地底美食之旅！



### ⚙️ 第二步：机械传动与大工业萌芽

* 基础发电与传动：依靠 [机械动力（Create）](https://www.mcmod.cn/class/2021.html) 开启大机器时代。
* 进阶机械扩展：
* **铁路网络**：通过 [汽鸣铁道](https://www.mcmod.cn/class/8230.html)、[推进工程](https://www.mcmod.cn/class/19600.html) 与 [铁路导航](https://www.google.com/search?q=https://www.mcmod.cn/class/14600.html)，在世界地图上规划火车调度与铁轨轨迹。
* **重力与物理**：结合 [瓦尔基里天空](https://www.google.com/search?q=https://www.mcmod.cn/class/1528.html) 与 [Clockwork](https://www.mcmod.cn/class/13550.html)，利用物理引擎组装出真正的车辆、轮船与飞艇。
* [CC: Tweaked](https://www.mcmod.cn/class/1681.html)：通过 Lua 编程控制机械和红石流程。



### 🔬 第三步：现代化能源、数字化存储与终极自动化

* **电气革命与重工业**：引入 [沉浸工程](https://www.mcmod.cn/class/463.html) 搭建大型多方块机器与高压电网，随后迈向 [通用机械 (Mekanism)](https://www.mcmod.cn/class/187.html) 的五倍产矿、高效发电机与终极数字化矿石开掘系统。
* **数字化仓储**：使用 [应用能源2 (AE2)](https://www.mcmod.cn/class/260.html) 搭建现代化量子存储网络。借助 [蓝图加农炮处理样板](https://www.google.com/search?q=https://www.mcmod.cn/class/25738.html) 与 [Schematic Energistics](https://www.google.com/search?q=https://www.mcmod.cn/class/23365.html)，将机械动力的投影技术与 AE2 自动合成完美结合，实现基地的“一键无缝扩建”。

### ⚔️ 第四步：地底深渊、异星战场与基地防卫

* **未知领域探险**：携带 [永恒枪械工坊](https://www.mcmod.cn/class/14980.html) 的现代武器，深入危险重重的 [Alex 的洞穴](https://www.mcmod.cn/class/12592.html) 探索史前或未来生态，或挑战 [暮色森林](https://www.mcmod.cn/class/61.html) 与 [太空探索](https://www.mcmod.cn/class/7395.html)。
* **全面战事防卫**：利用 [机械动力：火炮技术](https://www.mcmod.cn/class/18707.html) 与 [Mekanism 炮塔与铁丝网](https://www.google.com/search?q=https://www.mcmod.cn/class/15247.html) 构筑现代防线，让你的军工厂输出最大火力。
